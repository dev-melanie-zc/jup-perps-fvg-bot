FVG TERMINAL
Created by Melanie Zoe
This tool detects Fair Value Gaps (FVGs) across multiple timeframes on Jupiter Perps — SOL, ETH, and WBTC — using live data from Birdeye (the same source as jup.ag/perps).
Zone signals highlight potential discount (long) and premium (short) setups based on ICT/SMC market structure principles.
